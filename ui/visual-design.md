## 📱 状态栏显示方案

### 方案 A：完整模式
```
┌─────────────────────────────────────────────────┐
│  10月24日 周四 农历九月廿二 14:00:05            │
└─────────────────────────────────────────────────┘
```
- **适用场景**：大屏幕、状态栏空间充足
- **宽度**：约 280px

### 方案 B：标准模式
```
┌──────────────────────────────┐
│  10月24日 周四 14:00         │
└──────────────────────────────┘
```
- **适用场景**：一般使用
- **宽度**：约 160px

### 方案 C：精简模式
```
┌──────────────────┐
│  10/24 14:00     │
└──────────────────┘
```
- **适用场景**：状态栏拥挤时
- **宽度**：约 100px

### 方案 D：极简模式
```
┌────────────┐
│  14:00     │
└────────────┘
```
- **适用场景**：极度简洁
- **宽度**：约 60px

## 🎭 交互状态流程图

### 状态栏点击流程

```
   用户点击状态栏
         │
         ├─→ Popover 未显示
         │        │
         │        └─→ 显示 Popover (带动画)
         │             • Spring 动画
         │             • 从上向下滑入
         │             • 淡入效果
         │
         └─→ Popover 已显示
                  │
                  └─→ 隐藏 Popover (带动画)
                       • 淡出效果
```

### 日期切换流程

```
   用户点击 < 或 > 按钮
         │
         ├─→ 点击 <
         │     │
         │     └─→ 当前月份 -1
         │          • 日历网格左滑退出
         │          • 新网格从右侧滑入
         │          • 持续 0.3s
         │
         └─→ 点击 >
               │
               └─→ 当前月份 +1
                    • 日历网格右滑退出
                    • 新网格从左侧滑入
                    • 持续 0.3s
```

## 📐 响应式规则

### 多屏幕适配

```
┌─────────────────────────────────────────┐
│  主屏幕                 │  副屏幕       │
│                         │               │
│  [MenuBar] ←触发        │  [MenuBar]    │
│     ↓                   │               │
│  [Popover]              │               │
│  显示在主屏幕            │               │
└─────────────────────────────────────────┘

规则：Popover 始终显示在触发它的屏幕上
```

### 边缘检测

```
情况 1：正常位置
┌─────────────────┐
│  [MenuBar]      │
│     ↓           │
│  ┌─────────┐    │
│  │ Popover │    │
│  └─────────┘    │
└─────────────────┘

情况 2：靠近右边缘
┌─────────────────┐
│      [MenuBar]  │
│         ↓       │
│    ┌─────────┐  │
│    │ Popover │  │
│    └─────────┘  │
└─────────────────┘
Popover 向左偏移，避免被裁切
```
