# 状态栏日历 macOS 应用

一个优雅的 macOS 状态栏日历应用，具有完整的农历、节气和法定节假日支持。

## ✨ 特性

### 核心功能

- 🗓️ **完整日历视图** - 月历显示，支持快速导航
- 🌙 **农历支持** - 显示农历日期、天干地支、生肖
- 🎋 **二十四节气** - 准确标注传统节气（立春、春分等）
- 🎉 **法定节假日** - 标注中国法定休息日和调休补班
- 📜 **黄历功能** - 每日宜忌、冲煞、五行、建除
- ⚙️ **高度自定义** - 灵活配置状态栏显示内容

### 设计特色

- 🎨 **原生设计** - 完美融入 macOS 系统风格
- 🌗 **毛玻璃质感** - backdrop-blur 半透明效果
- ⚡ **流畅动画** - Motion 库实现的平滑过渡
- 🖱️ **灵活交互** - 支持悬停或点击触发
- 📱 **响应式** - 适配不同屏幕尺寸
- ♿ **无障碍** - 完整的可访问性支持（规划中）

## 🛠️ 技术栈

### 原型演示（Web）

- **语言**: TypeScript
- **框架**: React 18
- **构建工具**: Vite
- **样式**: Tailwind CSS
- **UI 组件**: Radix UI + Material-UI
- **动画**: Motion (Framer Motion)
- **农历**: lunar-javascript

- **语言**: Swift 5.5+
- **框架**: SwiftUI, AppKit
- **最低系统**: macOS 12.0+
- **开发工具**: Xcode 14+
