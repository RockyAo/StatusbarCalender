# 状态栏日历 macOS 应用

一个优雅的 macOS 状态栏日历应用，具有完整的农历、节气和法定节假日支持。

## ✨ 特性

- 🗓️ **完整日历视图** - 月历显示，支持快速导航
- 🌙 **农历支持** - 显示农历日期、天干地支、生肖
- 🎋 **二十四节气** - 准确标注传统节气
- 🎉 **法定节假日** - 标注休息日和调休补班
- 🎨 **原生设计** - 完美融入 macOS 系统
- 🌗 **深色模式** - 自动适配系统外观
- ⚡ **高性能** - 流畅动画，低资源占用
- ♿ **无障碍** - 完整的 VoiceOver 支持

## 💡 设计理念

打造具有「原生感」的 macOS 状态栏应用，如同系统自带功能的高级进化版：

- 使用系统原生字体 SF Pro 和 SF Symbols
